# Changelog

## 0.1.0

- Initial release of the Njord Mission panel.
- Leaflet map centred on the boat's live GPS position (~1 km view) with a boat
  marker that follows new fixes.
- GPS waypoint management: add/delete/clear waypoints with optional labels,
  numbered markers, and a mission-path polyline.
- Mission control: publish waypoints (Launch Mission) and abort (Stop Mission).
- Live mission-status indicator (idle / running / completed / aborted).
